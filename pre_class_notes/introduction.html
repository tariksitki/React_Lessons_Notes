To create a react app, you can aslo use Create React App (CRA) boilerplate. 1. Create React App is an officially supported way to create single-page React applications. It offers a modern build setup with no configuration. You can install create-react-app globally with the node package manager (npm). npm install -g create-react-app (once) Then run the generator in your chosen directory create-react-app my-app cd my-app/ npm start If you don't want to install it globally, you can use npx.
